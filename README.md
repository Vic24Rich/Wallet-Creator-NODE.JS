# Wallet-Creator-NODE.JS

### INSTALLATION
- Intall the Node.JS in [LINK](https://nodejs.org)
- Later install the PACKs that will be used:

| Name          |   Version   |
| ------------- | ----------- |
| BIP39         | Most recent |
| BIP32         | 2.0.6       |
| Bitcoinjs-lib | Most recent |

Sample:
```js
npm install bip39 and etc...
```
### RUNNING
```js
node .\createwallet.js
```

## THAT'S IT
